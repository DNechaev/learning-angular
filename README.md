# Learning Angular



## Install
``` 
cd ./server
npm install

cd ../client
npm install
``` 

## Build
```
cd ./client
ng build 
```

## Use

**Docker**
``` 
docker-compose up
```

**App**
``` 
Goto url: http://localhost:8090/
``` 

**App Api**
``` 
Goto url: http://localhost:8090/api/..
``` 

**Adminer**
```
host: localhost:8091
user: root
password: root
```

**Mysql**
``` 
host: localhost
port: 8092
user: root
password: root
```
